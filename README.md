# pcb-support
printed circuit board (PCB) support



I made a printed circuit board (PCB) support, with the objective to organize the projects is that i developing or testing. I can leave this on a piece of wood and attach it anywhere: walls, suitcases, tables, etc...

<br>

I believe that the better use of this support is for boards in teste fase, Where I don't need to move it much, just fix it in place and set the measurement and log points. So I don't recommend this suporte to work with welding for example.
